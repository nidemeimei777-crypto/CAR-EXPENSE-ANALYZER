# CAR-EXPENSE-ANALYZER
Приложение для учета расходов на автомобиль с анализом эффективности, разработанное с помощью REST-сервисов с использованием фреймворка FastAPI

## Функциональность

- CRUD автомобилей, заправок, расходов, напоминаний
- Регистрация и JWT-аутентификация
- Расчёт среднего расхода топлива (л/100 км)
- Расчёт стоимости одного километра пробега
- Сравнение эффективности двух автомобилей
- Автоматические напоминания о ТО (через заданный пробег)

## Технологии

- FastAPI
- Pydantic
- JWT (PyJWT, passlib)
- Хранение данных: CSV (или PostgreSQL через Docker)
