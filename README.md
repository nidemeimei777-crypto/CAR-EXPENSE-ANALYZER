# Car Expense Analyzer

Приложение для учёта расходов на автомобиль с анализом эффективности.  
Позволяет вести учёт топлива, ремонтов, страховок, налогов, штрафов и других трат.  
Автоматически рассчитывает средний расход топлива, строит круговые диаграммы расходов по категориям и сравнивает автомобили по эффективности.

## 🚀 Технологии

- FastAPI
- SQLAlchemy (async)
- PostgreSQL
- Docker / Docker Compose
- Jinja2 (HTML-шаблоны)
- JWT-аутентификация
- Pytest (тесты)

## 📋 Функциональность

- Регистрация и аутентификация (JWT)
- Управление автомобилями (CRUD)
- Учет расходов (ремонт, мойка, страховка, налоги, штрафы, платные дороги)
- Учет заправок (автоматическое создание расхода в категории «заправка»)
- Учет напоминаний о ТО
- Расчёт среднего расхода топлива (л/100 км)
- Сравнение автомобилей по эффективности (₽/100 км)
- Круговая диаграмма распределения расходов по категориям
